# AI Usage Disclosure

This project was developed with the assistance of generative AI tools (Claude by Anthropic). In the spirit of transparency and in compliance with course policies, this document details how AI was used throughout the development process.

## How AI Was Used

**Planning & Roadmap Generation**
The implementation roadmap (`ROADMAP.md`) was drafted with AI assistance to outline the project phases, identify potential pitfalls, and structure a logical commit history.

**Code Snippet Generation**
AI was used to generate initial code snippets for specific features, including session management logic, event delegation patterns, and DOM manipulation routines. These snippets served as starting points and were adapted, modified, or rewritten to fit the project's architecture and requirements.

**Debugging Assistance**
When encountering bugs during development, AI was consulted to help diagnose root causes and suggest potential fixes. The bugs themselves were identified and documented independently through manual testing, as reflected in the `BUGTRACKER.md` log.

**Optimization Recommendations**
AI provided suggestions for improving performance, such as debounced saving strategies and efficient DOM update patterns. These recommendations were evaluated on a case-by-case basis before being incorporated.

**Code Linting Overviews**
AI was used to review code for style consistency, potential issues, and adherence to best practices such as JSDoc documentation and error handling conventions.

## What Was Done Independently

- All final implementation decisions, including which AI suggestions to accept, reject, or modify
- Manual testing and cross-page verification across browsers
- Bug discovery, reproduction, and documentation (`BUGTRACKER.md`)
- Architectural decisions regarding class structure, data persistence strategy, and widget injection approach
- All final code was personally reviewed, understood, and committed

## Guiding Principle

AI served as a development aid comparable to consulting documentation or reference examples. Every piece of generated output was critically evaluated before adoption, and no code was included in the final submission without a thorough understanding of its purpose and behavior.
